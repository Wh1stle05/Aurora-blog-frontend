# Aurora Blog - 现代化极光风格技术博客

这是一个基于 React 19 + FastAPI 构建的高级现代化博客系统，采用了 **Aurora (极光)** 设计语言，结合毛玻璃效果 (Glassmorphism) 与灵动的物理交互。

## 🏗️ 项目架构 (Modern Directory Structure)

前端采用了高度可扩展且模块化的目录结构：

```text
src/
├── assets/             # 静态资源 (图片, 字体等)
├── components/
│   ├── common/         # 高度复用的基础原子组件
│   ├── layout/         # 全局布局组件 (Header, Footer, Toast, Container等)
│   └── features/       # 业务功能模块块
│       ├── auth/       # 认证模块 (登录, 注册弹窗)
│       ├── blog/       # 博客核心功能 (评论区, 代码高亮等)
│       └── home/       # 首页特有组件
├── context/            # 全局状态上下文 (如 ToastContext)
├── hooks/              # 自定义 React Hooks
├── pages/              # 页面级容器组件
├── services/           # API 接口抽象层
├── styles/             # 全局样式与常量
└── utils/              # 通用工具函数
```

## 🛠️ 核心技术栈

### 前端 (Frontend)
- **React 19**: 响应式 UI 框架
- **Framer Motion**: 驱动全站的丝滑物理动画
- **React Router 7**: 现代路由管理
- **PrismJS**: 实现动态切换主题的代码高亮
- **CSS Modules**: 解决样式冲突，实现组件化样式

### 后端 (Backend)
- **FastAPI**: 高性能 Python 异步 Web 框架
- **SQLAlchemy 2.0**: 数据库 ORM 映射
- **PostgreSQL**: 企业级关系型数据库
- **PyJWT**: 基于 Token 的安全认证系统

## ✨ 交互亮点

1. **沉浸式首屏**: 欢迎语垂直居中，背景自带缓慢流动的极光动效。
2. **毛玻璃 UI**: 全站采用 `backdrop-filter: blur`，营造高级的通透感。
3. **自适应高亮**: 代码块不仅支持语法高亮，还会随着全站深浅模式切换自动更换高亮主题。
4. **自研通知系统**: 位于导航栏下方的非遮挡式 Toast 提示，自带倒计时进度条。
5. **智能浏览量统计**: 解决了开发环境双发请求导致的浏览量误增问题。
6. **楼中楼评论**: 支持无限层级评论，并对超过 3 条的回复进行智能折叠。

## 🚀 快速开始

### 前端开发
```bash
cd blog4/frontend
npm install
npm run dev
```

### 后端开发
```bash
cd blog4/backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
